Latest PR is here: 
https://github.com/furkanYanteri1/vue-sm-platform/pull/10

### Currently live here:
https://wwww.furkanyanteri1.github.io/vue-sm-platform/

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests

```sh
npm run test:unit
```

### Lint with 

```sh
npm run lint
```
